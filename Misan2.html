By the way, do you know how the internet works?

Well, from my perspective, the internet is just the communication between a web browser and a server.The browser requests for information,while the server responds(with an html file i think)

As they communicate, Hyper text transfer protocol secure(HTTPS), ensures that what ever information being transfered is encrypted.Making it hard for intruders to snoop around i guess

Also, every website has an internet protocol address(IP).This address can be found in a domain naming system(DNS),which is like a dictionary for IP addresses

HTML,CSS & JAVASCRIPT, what do they mean?

Html stands for hypertextmarkup language, which is like the base structure of ever website created.
Css, known as cascading style sheets is used to beautify a website.To make the website more presentable.
Java script is responsible for bringing a website to life,it is responsible for allowing a website to be interactive with the user.
